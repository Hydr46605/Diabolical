# Optional Resource Pack

Optional item visuals and Ledger glyph markers. Gameplay works without it, and
contract names/actions remain plain text.

Build:

```sh
scripts/package_resourcepack.sh 0.5.0-dev
```

Install the zip as a resource pack, not a data pack.

## Included Items

- Infernal Primer
- Infernal Ledger
- Ashen Charm
- Midas Coin
- Hellbound Compass
- Awakened Hellbound Compass
- Black Writ
- Court Candle
- Bailiff Bell
- Sinners Effigy
- Ashen Brief
- Accusers Seal

## Ledger Glyphs

The pack includes a small bitmap font for contract markers:

- Pact of Embers
- Pact of Hunger
- Red Ledger Minor
- Pact of Copper
- Greed Clause
- The Red Ledger
- Blood Tithe
- Ashen Credit
- Grave Collateral

Textures are 16x16. Vanilla items keep normal models unless they carry
Diabolical custom model data.
