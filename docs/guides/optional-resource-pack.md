# Optional Resource Pack

Diabolical works without custom visuals. The optional pack only changes how the
Primer, Ledger, and relic items look.

Build it with:

```sh
scripts/package_resourcepack.sh 0.3.0-dev
```

Install the zip as a normal resource pack. Do not place it in the world's
`datapacks/` folder.

## Included Items

- Infernal Primer
- Infernal Ledger
- Ashen Charm
- Midas Coin
- Hellbound Compass
- Awakened Hellbound Compass

The textures are 16x16 and selected by namespaced custom model data strings.
Normal vanilla items still fall back to their normal models.
