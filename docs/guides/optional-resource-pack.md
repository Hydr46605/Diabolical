# Optional Resource Pack

Optional item visuals only. Gameplay works without it.

Build:

```sh
scripts/package_resourcepack.sh 0.4.2-rc1
```

Install the zip as a resource pack, not a data pack.

## Included Items

- Infernal Primer
- Infernal Ledger
- Ashen Charm
- Midas Coin
- Hellbound Compass
- Awakened Hellbound Compass
- Black Writ
- Court Candle
- Bailiff Bell
- Sinners Effigy
- Ashen Brief
- Accusers Seal

Textures are 16x16. Vanilla items keep normal models unless they carry
Diabolical custom model data.
