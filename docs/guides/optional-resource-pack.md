# Optional Resource Pack

Diabolical works without custom visuals. The optional pack only changes how the
Primer, Ledger, relics, and court items look.

Build it with:

```sh
scripts/package_resourcepack.sh 0.4.1-dev
```

Install the zip as a normal resource pack. Do not place it in the world's
`datapacks/` folder.

## Included Items

- Infernal Primer
- Infernal Ledger
- Ashen Charm
- Midas Coin
- Hellbound Compass
- Awakened Hellbound Compass
- Black Writ
- Court Candle
- Bailiff Bell
- Sinners Effigy
- Ashen Brief
- Accusers Seal

The textures are 16x16, and 0.4.1 refreshes the older icons while adding the
Sinbound Court set. Normal vanilla items still fall back to their normal models.
