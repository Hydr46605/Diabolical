# Infernal Ruins

Infernal Ruins are small Diabolical markers that can appear in newly generated
overworld chunks. They are discovery flavor, not required progression.

Current ruins:

| Ruin | What It Is |
| --- | --- |
| Ash Circle | a scorched ritual ring with a small cache |
| Ledger Cairn | a broken reading shrine |
| Collector Post | a basalt warning marker |

The caches use Diabolical loot tables with vanilla items such as paper,
blackstone, fire charges, bones, redstone, gold nuggets, and rare echo shards.

Existing chunks will not change. Test structure density in a fresh world or in
newly explored chunks before using it on a live server.
