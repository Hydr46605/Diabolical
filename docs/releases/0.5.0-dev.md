# Diabolical 0.5.0-dev

Red Ledger Expansion release.

## Highlights

- Adds Blood Tithe, Ashen Credit, and Grave Collateral contracts.
- Groups Ledger contracts into Starter, Red, and Major clauses.
- Adds active payment rows for the new contracts.
- Adds optional Ledger title overlay and per-contract item textures.
- Extends validation for contract routes, Ledger glyphs, item models, and release
  packages.

## Assets

- `Diabolical-0.5.0-dev.zip` - data pack
- `Diabolical-optional-resources-0.5.0-dev.zip` - optional resource pack
